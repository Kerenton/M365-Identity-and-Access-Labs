# Microsoft 365 Identity & Access Labs

## Overview
This repository documents hands-on identity and access management labs
built using Microsoft 365, Microsoft Entra ID, and Conditional Access.

The focus of this repository is enterprise identity design, access control,
and secure authentication rather than basic administration.

---

## Technologies & Services
- Microsoft Entra ID (Azure AD)
- Conditional Access
- MFA & Identity Protection
- SAML 2.0 Single Sign-On
- Custom Domains
- Microsoft 365 Security & Identity

---

## Labs Included
- Entra ID dynamic group design and rules
- Conditional Access baseline and custom policies
- Salesforce SAML SSO integration
- Custom domain configuration in Entra ID
- Identity troubleshooting and validation

---

## Design Principles
- Least privilege access
- Zero Trust authentication
- Policy-based access control
- Enterprise-ready identity patterns

---

## Who This Repo Is For
- Identity & Access Management roles
- Microsoft 365 administrators
- Security engineers transitioning into IAM
